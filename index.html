联想SR550安装RHEL6.7

发表于 2020-12-29

官网最低支持6.9系统，经过尝试，6.7也可以安装成功，但是需要自己编译安装对应网卡驱动。

设置磁盘阵列
设置过程参考文档：
链接: https://pan.baidu.com/s/1aflbTxgRsVPCV6J8GG6bcQ 提取码: tehh

具体过程可能不太一样，参考大概流程即可。

阵列卡驱动拷入U盘
阵列卡驱动下载地址：https://download.lenovo.com/servers/mig/2019/04/10/19877/lnvgy_dd_sraidmr_07.707.04.00_rhel6_x86-64.tgz

下载后解压，进入disks目录，选择对应系统版本的iso.gz文件解压，将解压后的dd.iso拷入U盘备用。

安装CentOS6.7
这里使用外置光驱加载之前刻录的系统盘进行安装。同时插入存有阵列卡驱动的U盘。
安装过程参考文档:
链接: https://pan.baidu.com/s/1TjK8ElNCKRV2Qr6WkCZ91A 提取码: b1fn

设置启动模式
系统安装成功后重启，无法启动系统，需修改启动模式为legecy mode。F1进入设置，进入BootManager，找到相关项修改即可。
修改后重启即可进入系统。

安装网卡驱动
联想官网提供了一键安装的网卡驱动，但是最低支持6.9版本。6.7版本的驱动需要自己下载编译安装。

在intel官网搜索对应网卡的驱动，这里搜索X722，找到相关驱动，选择一个较旧的版本，经测试2.8.43版本可用，下载地址：
https://downloadcenter.intel.com/zh-cn/download/28942/-Linux-PCIe-40-
下载后解压，参考readme编译安装即可。
需要注意的是,安装前确认系统kernel-headers、kernel-firmware、kernel-devel三个软件包已经安装，且版本一定要相同，以及编译工具gcc、gcc-c++已经安装，如果没有安装，请配置正确的yum源后使用yum命令来安装。

安装完成后重启，即可进行相关ip配置。

centos7安装docker

[root@bogon ~]# cat /etc/redhat-release
CentOS Linux release 7.5.1804 (Core) 

[root@bogon ~]# uname -a
Linux bogon 3.10.0-862.el7.x86_64 #1 SMP Fri Apr 20 16:44:24 UTC 2018 x86_64 x86_64 x86_64 GNU/Linux

卸载旧版本及依赖包

[root@bogon ~]# yum remove docker docker-client docker-client-latest docker-common docker-latest docker-latest-logrotate docker-logrotate docker-engine

设置docker存储库（Repository）

[root@bogon ~]# yum install -y yum-utils device-mapper-persistent-data lvm2

设置稳定存储库

[root@bogon ~]# yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo

开启或关闭测试存储库

yum-config-manager --enable docker-ce-test
yum-config-manager --disable docker-ce-test

(2)安装Docker CE

[root@bogon ~]# yum list docker-ce --showduplicates |sort -r

[root@bogon ~]# yum install docker-ce -y

启动Docker服务并配置开机自启动

[root@bogon ~]# systemctl start docker
[root@bogon ~]# systemctl enable docker

测试Docker是否安装成功

[root@bogon ~]# docker run hello-world

Linux之CentOS7 安装Tomcat9

发表于 2020-08-18 更新于 2021-05-18

#Linux 之 CentOS 7 安装Tomcat9

1、下载Tomcat

tomcat9下载地址：https://tomcat.apache.org/download-90.cgi

2、上传到服务器中

推荐使用xftp上传

3、安装tomcat

3.1、解压并复制到 /usr/tomcat 中

cp apache-tomcat-9.0.37.tar.gz /usr/tomcat9

tar zxvf tomcat9

3.2、设置环境变量

vim /etc/profile

# tomcat9 
export CATALINA_HOME=/usr/tomcat9
export PATH=${CATALINA_HOME}/bin:$PATH

source /etc/profile

![](https://img2020.cnblogs.com/blog/1480764/202007/1480764-20200714164859665-2055316445.png)

3.3、配置catalina.sh文件

vim /usr/tomcat9/bin/catalina.sh

在头部添加

JAVA_HOME=/usr/jdk1.8.0_251
JRE_HOME=${JAVA_HOME}/jre

3.4、启动tomcat服务

./startup.sh

3.5、放行8080端口

#放行8080端口
firewall-cmd --zone=public --add-port=8080/tcp --permanent
#重启防火墙
firewall-cmd --reload

3.6、打开浏览器访问测试

4.2、增加 manager-gui 图形化管理界面的访问权限

复制代码
vim /usr/tomcat9/conf/tomcat-users.xml
#添加如下代码
<role rolename="manager-gui"/>
<user username="tomcat" password="tomcat" roles="manager-gui"/>

修改webapps/manager/META-INF目录下的context.xml,在allow行的末尾加上|\d+.\d+.\d+.\d+表示允许所有主机访问。

<Context antiResourceLocking="false" privileged="true" >
<Valve className="org.apache.catalina.valves.RemoteAddrValve"
     allow="127\.\d+\.\d+\.\d+|::1|0:0:0:0:0:0:0:1|\d+\.\d+\.\d+\.\d+" />
<Manager sessionAttributeValueClassNameFilter="java\.lang\.(?:Boolean|Integer|Long|Number|String)|org\.apache\.catalina\.filters\.CsrfPreventionFilter\$LruCache(?:\$1)?|java\.util\.(?:Linked)?HashMap"/>

重启tomcat生效

4.3、配置tomcat开机自启

在/usr/lib/systemd/system目录下增加tomcat9.service文件，内容如下：

[Unit]

Description=Tomcat

After=syslog.target network.target remote-fs.target nss-lookup.target



[Service]

Type=oneshot

ExecStart=/usr/tomcat9/bin/startup.sh

ExecStop=/usr/tomcat9/bin/shutdown.sh

ExecReload=/bin/kill -s HUP $MAINPID

RemainAfterExit=yes


[Install]

WantedBy=multi-user.target

退出并保存，执行systemctl enable tomcat9.service使tomcat9随开机启动

systemctl enable tomcat9.service
systemctl start tomcat9.service    启动tomcat

systemctl stop tomcat9.service    关闭tomcat   

systemctl restart tomcat9.service    重启tomca

Linux 之 CentOS 7安装MySQL5.7

发表于 2020-08-18

#Linux 之 CentOS 7安装MySQL5.7

1.下载MySQL yum 包

wget http://repo.mysql.com/mysql57-community-release-el7-10.noarch.rpm

2.安装MySQL源

rpm -Uvh mysql57-community-release-el7-10.noarch.rpm

3.安装MySQL服务端

yum install -y mysql-community-server

4.启动MySQL

systemctl start mysqld.service

5.检查是否启动成功

systemctl status mysqld.service

6.获取临时密码

mysql5.7为root用户随机生成了一个密码

grep 'temporary password' /var/log/mysqld.log

7.登录mysql，修改密码

mysql -uroot -p

8.修改密码

更改密码规则

mysql> set global validate_password_policy=0;
mysql> set global validate_password_length=1;

设置密码

ALTER USER 'root'@'localhost' IDENTIFIED BY 'yourpassword';

9.开启远程连接

mysql> use mysql;
#%代表所有连接
mysql> update user set Host = '%' where Host = 'localhost' and User='root';
#刷新权限表
mysql> flush privileges;

10.开启开机自启

一般默认自启

systemctl enable mysqld
systemctl daemon-reload

11.设置字符集

打开配置文件

vim /etc/my.cnf

改成如下

# For advice on how to change settings please see
# http://dev.mysql.com/doc/refman/5.7/en/server-configuration-defaults.html

[mysql]
default-character-set=utf8

[mysqld]
datadir=/var/lib/mysql
socket=/var/lib/mysql/mysql.sock
default-storage-engine=INNODB
character_set_server=utf8

symbolic-links=0

log-error=/var/log/mysqld.log
pid-file=/var/run/mysqld/mysqld.pid

重启mysql，使配置生效

service mysqld restart

12.放行3306端口

#放行3306端口
firewall-cmd --zone=public --add-port=3306/tcp --permanent
#重启防火墙
firewall-cmd --reload

13.卸载mysql仓库

14.navicat连接测试

使用navicat创建数据库，创建表，添加数据。服务端测试查询

Linux 之 CentOS 7 安装JDK1.8

发表于 2020-08-17 更新于 2020-12-31

#Linux 之 CentOS 7 安装JDK1.8

1、卸载系统自带的openJDK和相关JAVA文件

1.1、查看是否带有openJDK

java -version

1.2、查询java文件

rpm -qa | grep java

1.3、卸载openJDK1.8

rpm -e --nodeps java-1.8.0-openjdk-headless-1.8.0.131-11.b12.el7.x86_64
rpm -e --nodeps java-1.8.0-openjdk-1.8.0.131-11.b12.el7.x86_64

2、下载JDK1.8Linux版

可以在windows下下好压缩包，使用xftp传到Linux上

3.1、复制到 /usr/ 目录下

cp jdk-8u271-linux-x64.tar.gz /usr/local/src

3.2、进入到/usr/local/src 目录下 并解压

tar zxvf jdk-8u271-linux-x64.tar.gz

3.3、配置JDK环境变量

在vim /etc/profile 文件配置环境变量

export JAVA_HOME=/usr/local/src/jdk1.8.0_271
export CLASSPATH=$:CLASSPATH:$JAVA_HOME/lib/
export PATH=$PATH:$JAVA_HOME/bin

使配置文件生效

生效命令：source /etc/profile

4、测试

java -version
java
javac

#一 安装前准备

1、检查是否已经安装过mysql，执行命令

[root@localhost /]# rpm -qa | grep mysql

从执行结果，可以看出我们已经安装了mysql-libs-5.1.73-5.el6_6.x86_64，执行删除命令

[root@localhost /]# rpm -e --nodeps mysql-libs-5.1.73-5.el6_6.x86_64

#再次执行查询命令，查看是否删除

[root@localhost /]# rpm -qa | grep mysql

#查看服务器残留mysql数据并删除

[root@localhost ~]# rpm -qa |grep -i mysql

[root@localhost ~]# find / -name mysql

  /usr/share/mysql

#解压tar包

[root@localhost ~]# cd /opt
[root@localhost opt]# tar jxvf mysql-8.0.21-linux-glibc2.12-x86_64.tar.xz

#mv到/usr/local/目录下，并命名为mysql

[root@localhost opt]# mv mysql-8.0.21-linux-glibc2.12-x86_64 /usr/local/mysql

#创建mysql配置存放目录data

[root@localhost opt]# cd /usr/local/mysql
[root@localhost mysql]# ll
[root@localhost mysql]# mkdir data

#创建mysql用户组和用户

[root@localhost mysql]# groupadd mysql
[root@localhost mysql]# useradd -r -g mysql mysql

#为mysql目录修改用户权限

[root@localhost mysql]# chown -R mysql:mysql /usr/local/mysql/
